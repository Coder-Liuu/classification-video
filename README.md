# Classification：视频分类在Tensorflow当中的实现

## 所需环境
tensorflow-gpu=2.2.0

库的近似版本应该也可以

## 文件下载

默认数据集可以在百度云下载,下载完需要放在videos/目录中

链接：https://pan.baidu.com/s/1U_PZsce8g74y5e5rKzzRww   
提取码：sfn0


## 训练步骤

1. 运行create_image.py文件，生成视频采样特征。
2. 运行main.py文件，训练模型
3. 运行predict.py文件，进行预测
